# Social Media Data Analysis

This folder contains coursework from the Social Media Data Analysis class.  
Across the assignments, I worked with different kinds of online and platform-related data, including X (Twitter), YouTube, Reddit, Google Trends, and public datasets.

## Platforms and Data

| Assignment | Platform / Data                   | Type of data                                              |
| ---------- | --------------------------------- | --------------------------------------------------------- |
| 1          | Google Trends, World Bank, ILINet | Search behavior data, macro/public data, time-series data |
| 2          | X (Twitter), YouTube              | Social media posts, channel statistics, engagement data   |
| 3          | X (Twitter), YouTube, Vent        | Text data for sentiment and emotion analysis              |
| 4          | X (Twitter), Reddit               | Retweet network data, Reddit thread/comment structure     |

## Analyses

- **Assignment 1:** Analyzed the relationship between future-oriented search behavior and GDP, and modeled flu trends using Google Trends and ILINet data.
- **Assignment 2:** Examined social impact on X and YouTube using engagement statistics, regression, and bootstrapping.
- **Assignment 3:** Compared sentiment and emotion detection methods across social media text data.
- **Assignment 4:** Analyzed retweet networks of politicians on X and compared discussion tree structures between political and non-political subreddits on Reddit.

## Methods and Skills

- Data collection and preprocessing
- Social media and platform data analysis
- Regression analysis
- Sentiment analysis
- Emotion detection
- Network analysis
- Statistical testing
- Time series data analysis
